厨神篇

  • 2023-08-21
  • By.NoMrXu

从此刻开始，一名厨神踏上了征途。

基本功

拿刀

右手大拇指与食指夹住刀身，剩余三指握住刀把，保持稳定。

左手以中指第一指关节抵住刀身，指尖内敛。

蒜蓉生菜

  1. 生菜剥好洗净
  2. 热油，加入蒜末到微黄（可以早点，不然直接爆油）
  3. 倒入生菜，开超
  4. 加入盐（酱油）
  5. 一会会就能出锅

涨蛋

  1. 鸡蛋打散，加盐
  2. 热油（一定要热，冒冒烟
  3. 倒鸡蛋，随便超超
  4. 出锅

番茄炒蛋

  1. 鸡蛋打散（加点盐？
  2. 热油，倒鸡蛋
  3. 加番茄
  4. 加点水（一点点）
  5. 加盐、料酒、糖！！！
  6. 加葱
  7. 出锅

手撕包菜

  1. 横切切两半，手撕，加盐清洗
  2. 大蒜、辣椒切碎
  3. 热油，爆香
  4. 下锅，放醋
  5. 加入盐、糖、鸡精、酱油
